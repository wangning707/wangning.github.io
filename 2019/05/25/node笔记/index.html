node笔记

2019-05-25

node crypto包加密，支持rsa ，md5， res等加密算法

node sharp包 对图片进行处理 text-to-svg 文字图片化

node fs包 对文件进行读写处理，文件夹进行创建删除处理

node axios 发送请求，可以设置返回 stream 类型获取文件流并且用 .pipe(fs.createWrite(FileName))方法接受，传入到文件夹中

node multer 服务器接受文件

服务器路径默认为静态文件夹目录

服务器路径中不要含中文字符
